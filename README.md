# Chicken_Door
Forked from ericescobar/Chicken_Door, this repo includes scripts to open and close the chicken door through crontab (based on time of day), and control it through the internet.

# Known Packages Installed

-    nginx
-    MySQL
-    PHP
-    PHPMyAdmin
-    python-pip
